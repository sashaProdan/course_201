Inline level Elements

Layout

Inline level elements line up on the same line, one after another.

Width

Occupy only the width their content requires.

Height

Will expand and contract horizontally as necessary to accomodate its content.

Setting Height and Width Properties

Width and height properties set on inline elements are ignored.

Width: 50px, height: 50px - Google

Width: 150px, height: 70px - Google

Width: 80px, height: 10px - Google

Margins

Top and bottom margins are not accepted

Google Yahoo Yandex

Paddings

Paddings for inline level elements work on all four sides.
